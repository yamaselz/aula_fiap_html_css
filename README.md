# HTML
```
Sempre utilizar letra minuscula em codigo

Para comentar o codigo: <!-- --> (atalho ctrl ; em ptbr ou ctrl / americano)

H
    O html possui 06 niveis de tags para H
    tag h1 traz o titulo de maior relevância que provavelmente tem o nome da sua aplicação

    # H1 é o maior e os outros menores

    Boa prática é utilizar o H1 apenas uma vez (google tmb utiliza para indexar na pesquisa)

    # H2, H3, H4 muito utilizado

    # H5 e H6 pouco utilizado em conteúdo web

Todo conteúdo inserido em uma pagina deve estar dentro de tags, não pode haver conteúdo solto.

Tag paragrafo com conteúdo


Figure --> Container que recebe imagens audios e videos, serve para receber a imagem, dentro dele inserir a tag [img]
    o atributo alt de img trás assebilidade da pagina, é a descricao da imagem

    Dentro do figure é possivel inserir um figcaption, que serve de legenda para o tipo de imagem

Hyperlinks
    utilizando a tag [a] é possivel criar links,
    possivel customizar para abrir link em nova aba utilizando a propriedade [target = "_blank"]

    Para o Hyperlink em uma imagem é o mesmo processo, colocar entre a tag [a]

```
